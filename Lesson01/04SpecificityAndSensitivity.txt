SPECIFICITY AND SENSITIVITY.

EXAMPLE:  1% of the population has some cancer.
         90% of this people with cancer, test positive for some test T.
              ^SENSITIVITY. 
         90% of all the people without cancer, test negative for some test T.
              ^SPECIFICITY.


The percent of data that has some property X and test positive in some test T; is
called the SENSITIVITY.

The percent of data that doesn't have some property X and test negative in some
test T; is called the SPECIFICITY.
